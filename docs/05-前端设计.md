## 5. 核心组件设计

### 5.1 ChatPanel（对话面板）

对话面板是整个应用的核心交互组件，负责：

- 显示对话消息列表（支持文本和结构化数据）
- 消息输入和发送
- SSE 流式消息接收和逐字显示
- 操作栏（确认/修改/回退按钮，根据状态动态显示）

**状态管理（chatStore）：**
```typescript
interface ChatState {
  currentSession: ChatSession | null;
  messages: ChatMessage[];
  isStreaming: boolean;          // 是否正在接收流式消息
  streamingContent: string;     // 正在流式接收的内容
  currentState: ConversationState;
  availableActions: Action[];
  
  // Actions
  createSession: (projectId: number) => Promise<void>;
  sendMessage: (sessionId: number, message: string) => void;
  confirmStep: (sessionId: number) => void;
  revise: (sessionId: number, message: string) => void;
}
```

### 5.3 SqlEditor（SQL 编辑器）

基于 Monaco Editor 的 SQL 编辑器封装：

- SQL 语法高亮
- 只读模式（预览）和编辑模式
- 多 Tab 支持（每个步骤一个 Tab）
- 复制和导出功能

### 5.4 TableBrowser（表浏览器）

左侧面板，以树形结构展示项目下的所有表和字段：

```
▸ sales_order (订单表)
  ├─ order_id (BIGINT) - 订单ID
  ├─ user_id (BIGINT) - 用户ID
  ├─ amount (DECIMAL) - 金额
  └─ order_date (DATE) - 日期
▸ department (部门表)
  ├─ dept_id (INT) - 部门ID
  └─ dept_name (VARCHAR) - 部门名称
```

支持搜索、展开/折叠。

### 5.5 StepFlow（步骤流程）

使用 Ant Design 的 Steps 组件或自定义实现，展示作业步骤的线性流程：

```
[1. 关联表] ──→ [2. 汇总额] ──→ [3. 生成排名]
   ✅              🔄              ⏳
```

### 5.6 StructuredCard（结构化数据卡片）

对话中 AI 返回的结构化数据，用卡片形式展示：

- **TableRecommendCard**：表格形式展示推荐的表和字段，带勾选框
- **StepDesignCard**：步骤列表，每步显示名称、描述、类型
- **SqlBlock**：SQL 代码块，带语法高亮和复制按钮

## 6. 状态管理

使用 Zustand 进行轻量级状态管理，每个模块一个 store：

| Store | 职责 |
|-------|------|
| projectStore | 当前项目信息、项目列表 |
| metadataStore | 当前项目的表/字段元数据 |
| chatStore | 当前对话会话、消息列表、流式状态 |
| jobStore | 作业列表、当前作业详情 |

## 7. API 调用封装

### request.ts（Axios 实例）

```typescript
const request = axios.create({
  baseURL: '/api',
  timeout: 30000,
});

// 响应拦截器：统一处理错误
request.interceptors.response.use(
  (response) => {
    const { code, message, data } = response.data;
    if (code !== 200) {
      antdMessage.error(message);
      return Promise.reject(new Error(message));
    }
    return data;
  },
  (error) => {
    antdMessage.error('网络错误');
    return Promise.reject(error);
  }
);
```
